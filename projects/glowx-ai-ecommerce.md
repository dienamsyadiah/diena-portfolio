# GlowX AI E-Commerce

GlowX AI E-Commerce adalah sistem informasi e-commerce berbasis web untuk produk kecantikan. Project ini dilengkapi dengan halaman pengguna, dashboard admin, pengelolaan produk, analisis AI, rekomendasi AI, dan database SQL.

## Fitur Utama
- Halaman utama pengguna
- Dashboard admin
- Pengelolaan data produk
- Analisis berbasis AI
- Rekomendasi AI
- Riwayat analisis pengguna
- Database menggunakan SQL

## Teknologi
- PHP
- Composer
- MySQL
- HTML
- CSS
- JavaScript
